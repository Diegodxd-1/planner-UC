# 📖 Glosario de Términos - Planner-UC

## Términos Técnicos
*   **CSP (Constraint Satisfaction Problem)**: Problema matemático definido como un conjunto de objetos cuyo estado debe satisfacer una serie de restricciones.
*   **OR-Tools**: Suite de software de código abierto de Google para optimización combinatoria.
*   **CP-SAT**: Solver de programación con restricciones que utiliza satisfacción de circuitos y búsqueda perezosa. Es el motor principal de Planner-UC.
*   **MERN Stack**: Conjunto de tecnologías (MongoDB, Express, React, Node.js) utilizadas para el desarrollo de la aplicación web.
*   **FastAPI**: Framework web moderno y rápido para construir APIs con Python.

## Términos de Negocio (Académicos)
*   **Bloque**: Unidad mínima de tiempo académico (90 minutos).
*   **Sección**: Instancia específica de un curso que agrupa uno o más bloques semanales en un aula y horario determinados.
*   **Demanda Académica**: Número de estudiantes habilitados para llevar un curso según sus prerrequisitos aprobados.
*   **Currículo Flexible**: Modelo universitario donde el estudiante elige su ruta de aprendizaje, generando una demanda variable por ciclo.
*   **Oferta Académica**: Conjunto de secciones y horarios que la universidad pone a disposición antes del proceso de matrícula.
