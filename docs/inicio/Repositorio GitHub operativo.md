## 2. Descripción
Este repositorio contiene la estructura completa del proyecto "Planner UC", incluyendo:
* **docs/**: Documentación del proyecto en formato Markdown.
* **frontend/**: Código fuente de la interfaz de usuario (React.js).
* **backend/**: Lógica de negocio y API (Node.js + Express).
* **README.md**: Información general del proyecto.
